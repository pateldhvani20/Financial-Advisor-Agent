💰 Financial Advisor & Smart Expense Manager
📌 Project Overview

This project is an AI-powered Financial Advisor and Expense Manager that:

Extracts expenses from payment screenshots using OCR

Categorizes expenses automatically

Calculates budget distribution

Generates personalized financial advice

Uses structured financial principles instead of hallucinated advice

We are starting with Track A (Foundation Phase) and will upgrade to Track B after building a stable working system.

🎯 Core Input Parameters

Monthly Income

Fixed Expenses

Variable Expenses

Existing Savings

Existing Investments

Risk Appetite (Low / Medium / High)

Financial Goal

Time Horizon

Inflation Rate (future API integration)
